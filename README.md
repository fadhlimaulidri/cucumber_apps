# cucumber_apps

## Appium
### Precondition node & npm
1. install `nvm`
2. set default `nmv v6.11.2` => `$nvm alias default v6.11.2`
3. get `npm 3.10.10` => `$npm -v`
4. get `node v6.11.2` => `$node -v`
5. install appium `npm install -g appium`
6. `$ appium`
